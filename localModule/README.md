# F1 Bot local version

## How-to

1. download this file into a custom directory (either on PC or Smartphone)
2. Open the file using a preferred browser
3. Fetch data
4. Fill in the predictions
5. Save the predictions into a JSON file **into the same directory** (naming and download is automated)
6. Check the directory content
7. Next time, first fetch, then load

Score for each session type can be set on **line 100**
